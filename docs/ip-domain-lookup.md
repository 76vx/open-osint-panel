# IP / Domain Lookup Module

This module allows OSINT analysis of IP addresses and domains using public data.

## Features
- IP address lookup
- Domain resolution to IP
- Public data via ipinfo.io
- Legal and educational usage

## Endpoints
GET /api/osint/lookup/:target

Examples:
- /api/osint/lookup/8.8.8.8
- /api/osint/lookup/google.com
